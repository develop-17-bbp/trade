### 3.2 Paper-mode "trade-at-any-cost" gate softening

Operator directive: *"ACT must trade in Robinhood paper at any cost — initially with NO hard gates blocking trades. Real capital still strict."*

Implemented in `src/trading/conviction_gate.py`, `cost_gate.py`, `safe_entries.py`, `protections.py`, `executor.py`, `agentic_bridge.py`. Every gate now follows the same pattern:

```python
is_real_capital = os.environ.get("ACT_REAL_CAPITAL_ENABLED", "").strip() == "1"
if reject_condition and not is_real_capital:
    return SoftPass(reason=f"paper_advisory_{tag}", size_multiplier=0.25)
```

Gates softened (paper-mode only):
- TF 1h/4h misalignment, multistrategy weak, macro misaligned, Hurst non-trending, RANGING regime, longs-only-venue SHORT-veto, CALL/PUT hold-strategy mismatch, score below floor, throttle pause, protections-layer entry block, sniper min-confluence, cost-gate spread margin.

The brain's TradePlan is the sole authority. Authority rules from the operator's authority PDF (small body, news blackout, etc.) remain hard regardless of mode.

### 3.3 Brain-as-authority + autonomous tooling

**TICK_SNAPSHOT (`9213905`)** — every executor tick now writes the full signal vector (TF directions, RSI, ATR, sniper score, ML confidence, regime, S/R, orderbook imbalance, macro bias, recent trades, position state) to the agentic context BEFORE the LLM is invoked. Closes 11 brain blind spots audited at session start.

**Cross-tick analyst memory (`e4c38c7`)** — `read_recent_analyst_traces(asset, limit=5, max_age_s=900)` injected into the agentic seed so the analyst sees its own last 5 decisions for the asset. Currently it only saw scanner reports; now there is chain-of-reasoning continuity across ticks.

**Read-only Robinhood query tools (`e4c38c7`)** — operator directive: *"LLM must operate autonomously with read-only Robinhood API."* Four new tools registered with the brain's `ToolRegistry`:
- `query_robinhood_balance` — buying power + cash + status
- `query_robinhood_positions` — BTC/ETH holdings summary
- `query_robinhood_quote` — live bid/ask/mid/spread
- `query_recent_robinhood_fills` — recent live + paper fills

Strictly read-only. Brain proposes via `submit_trade_plan`; executor handles submission.

**Subsystem queries (`e4c38c7`)** — `query_12_agents` (full 13-fixed + transient-persona debate verdict) and `query_strategy_universe` (top-k positive-EV strategies for current regime) callable by the brain.

**PAPER DIRECT FIRE (`6b76850`)** — at the very top of `_evaluate_entry`, before any gate runs:

```
IF paper mode AND active TF signal AND no open position
THEN _paper.record_entry(direction=TF_majority, qty=1%_equity/price,
     sl=price ± 2*ATR, reasoning="paper-mode direct fire: TF signals {...}")
     AND return.
```

Real-capital path (`ACT_REAL_CAPITAL_ENABLED=1`) is completely unaffected — the early return is gated. Brain still runs in parallel agentic shadow path. Both paths emit trades; over time the brain's plans converge with rule-stack proposals as the soak counter accumulates.
